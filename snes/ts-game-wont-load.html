Game won't load

Game won't load

A cartridge that boots to a black screen, freezes on a logo, or shows scrambled graphics is one of the most common Super Nintendo complaints. The good news is that the cause is usually a dirty connection rather than a dead game.

Start with the contacts

  • Power off, remove the cartridge, and clean its edge connector with isopropyl alcohol on a swab.
  • Let it dry completely before reinserting; residual moisture can make things worse.
  • Clean the console's cartridge slot pins the same way if cleaning the cart alone does not help.

If contacts are clean

  • Try several reinsertions, seating the cartridge firmly and squarely.
  • Test another known good game to determine whether the fault is the cartridge or the console.
  • For original carts, a dead save battery does not stop a game booting, so look elsewhere for load failures.

Flashcart specifics

If you are using a flashcart, confirm the ROM file is intact, supported by the firmware, and that the SD card is healthy. A corrupted file or an unsupported enhancement chip can prevent a specific title from launching while others work fine. contacts black screen

Avoid blowing into cartridges, which adds moisture and corrodes contacts over time. Methodical cleaning and reseating resolve the large majority of load failures.
