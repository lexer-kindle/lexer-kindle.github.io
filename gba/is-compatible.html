Is my GBA compatible?

Is My GBA Compatible?

Good news: every Game Boy Advance model can run a flashcart, and all of them play the same game library. Where models differ is in screen mods, because the internal layout and the amount of free space inside the shell vary. The table below summarizes what each model supports.

ModelFlashcartsScreen modNotes
AGB-001 (original)YesIPS / backlight kitDim stock screen; the prime candidate for a brightness upgrade.
AGS-001 (front-lit SP)YesIPS kit, shell trim neededFront light is uneven; many owners upgrade the panel.
AGS-101 (backlit SP)YesAlready backlitBest stock screen; no kit required, only a transplant donor.
GBA MicroYesAlready backlitTiny, sharp screen; not GB/GBC backward compatible.

How to check quickly

  1. Find the model number printed on the back label.
  2. If it begins with AGB, you have the original tablet-style unit.
  3. If it begins with AGS, you have a clamshell SP.

Unsure which SP you have?

AGS-001 and AGS-101 look identical outside. The model identification page explains the power-button and menu tests that tell them apart.
